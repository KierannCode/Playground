virtual OSDL::Data* OSDL::Data::createCopy ( ) const
pure virtual

Creates a copy of the object.

This method returns a pointer to a new copy of the OSDL::Data object.
It is pure virtual and must be implemented in any derived class of OSDL::Data.
The copy is allocated with 'new' and must be manually destroyed using 'delete'.
For consistency, if DerivedClass is a derived class of OSDL::Data, its implementatio by default shall be as follows :

DerivedClass *DerivedClass::createCopy() {
return new DerivedClass(*this)';
}
